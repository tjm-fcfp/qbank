About this bank

A growing, peer-built question bank for the Fellowship (Family Medicine) Summative Viva — journal critique articles and PEL scenarios, each paired with probes and suggested answers kept hidden until you've attempted them.

Built to outlast any single cohort. Contributions from each batch keep it alive.

Adding material

Every article and scenario lives in the DATA object near the top of this file. To add one, copy an existing item, change the fields, and save. Longer reference notes (the in-app pages an item can link to from its reading list) live in the separate REFS object just after DATA — add a key there and point to it with { label, ref, note } in an item's reading list. No server or build step — open the file or host it anywhere static.
